**PDF Royale** is the ultimate tool for viewing, annotating, merging, and manipulating your PDFs directly in your browser. Experience it live at **[pdfroyale.com](https://pdfroyale.com)**. Built with a stunning modern interface and a focus on user privacy—everything runs locally. No subscriptions. No uploading documents to servers. 

## ✨ Key Features
- **🔒 100% Client-Side Privacy:** All rendering and operations happen locally using WebAssembly.
- **🖍️ Studio Annotations:** Draw naturally, add smart shapes, highlight text, and erase with infinite Undo/Redo.
- **📄 Document Management:** Combine multiple PDFs effortlessly or surgically remove pages.
- **🖼️ Seamless UI:** Fast, gorgeous glassmorphic design featuring infinite pan & zoom.

## 🚀 Getting Started

1. **Clone & Install:**
   ```bash
   git clone https://github.com/realms-overflow/pdf-editor.git
   cd pdf-editor
   npm install
   ```
2. **Run Locally:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000).
